# 정산

영수증 사진과 거래내역 엑셀을 넣으면, 소상공인의 신고 준비에 필요한 장부 초안과 홈택스 입력용 요약을 만들어주는 AI 정산 작업대입니다.

> 흩어진 지출 자료를 검토 가능한 장부와 신고 준비 문서 초안으로 바꿔, 세금 신고 전 가장 번거로운 정리 시간을 줄입니다.

## 왜 필요한가

소상공인과 프리랜서에게 세금 신고가 어려운 이유는 마지막 제출 버튼 때문만이 아닙니다. 진짜 병목은 그 전 단계입니다.

- 영수증은 사진첩에 흩어져 있습니다.
- 카드사와 플랫폼 거래내역은 엑셀로 따로 존재합니다.
- 장부에는 날짜, 거래처, 금액, 부가세, 증빙유형을 다시 옮겨야 합니다.
- 홈택스 입력 전에는 필요경비와 소득금액을 다시 확인해야 합니다.

`정산`은 이 반복 작업을 하나의 흐름으로 묶습니다. AI가 먼저 정리하고, 사람은 필요한 값만 검토합니다.

## 사용 흐름

1. 영수증 사진 또는 거래내역 엑셀을 업로드합니다.
2. AI와 파일 파서가 거래를 장부 카드로 정리합니다.
3. 사용자는 금액, 계정과목, 증빙유형을 확인하고 틀린 값만 고칩니다.
4. 확인된 거래만 필요경비 요약과 홈택스 입력용 요약에 반영됩니다.
5. 간편장부 엑셀과 신고 준비 문서 xlsx 초안을 다운로드합니다.

## 핵심 기능

- 영수증 이미지 인식 및 거래내역 엑셀 업로드
- 거래일, 거래처, 금액, 공급가액, 부가세, 계정과목, 증빙유형 추출
- 모바일에서도 보기 쉬운 카드형 검토 UI
- `확인` 버튼 기반 검토 완료 처리
- 필요경비 요약 및 총수입금액 기반 소득금액 계산
- 홈택스 입력용 요약 화면
- 간편장부 엑셀 다운로드
- `총수입금액 및 필요경비명세서`, `간편장부소득금액계산서` xlsx 초안 생성

## 우리가 잘라낸 것

`정산`은 홈택스 자동 제출 서비스가 아닙니다.

세금 신고 자동 제출은 인증, 보안, 법적 책임, 세무 판단 리스크가 큽니다. 그래서 MVP에서는 제출 버튼을 대신 누르는 대신, 실제 사용자가 먼저 막히는 `자료 정리 → 장부화 → 입력값 준비`를 해결합니다.

이 선택이 `정산`의 신뢰 전략입니다.

- 모르는 세금값은 억지로 채우지 않습니다.
- 사용자가 확인한 거래만 완료 처리합니다.
- 산출물은 제출본이 아니라 신고 전 검토용 초안입니다.
- 서버 데이터베이스와 로그인을 쓰지 않고 원본 파일을 저장하지 않습니다.

## 산출물

### 간편장부 엑셀

검토된 거래일, 거래처, 내용, 총액, 공급가액, 부가세, 계정과목, 증빙유형을 바탕으로 `jeongsan-simple-ledger.xlsx`를 생성합니다.

### 신고 준비 문서 초안

총수입금액과 검토된 장부 항목을 바탕으로 `jeongsan-tax-prep-draft.xlsx`를 생성합니다.

포함 시트:

- 홈택스 입력용 요약
- 총수입금액 및 필요경비명세서
- 간편장부소득금액계산서
- 확인 필요 항목

## 심사위원에게 보여줄 포인트

### 1. 실제 문제를 겨냥합니다

소상공인은 신고 버튼보다 신고 전 자료 정리에서 먼저 막힙니다. `정산`은 이 반복 업무를 줄입니다.

### 2. AI를 믿을 수 있게 씁니다

AI가 모든 값을 단정하지 않습니다. 검토 카드에서 사용자가 직접 확인하고, 확인된 값만 산출물에 반영합니다.

### 3. 데모가 끝까지 이어집니다

업로드에서 끝나지 않고 장부 카드, 필요경비 요약, 홈택스 입력용 요약, 문서 다운로드까지 이어지는 end-to-end 흐름을 보여줍니다.

## 한 문장으로

`정산`은 세금 신고를 대신하지 않습니다. 대신 소상공인이 신고 전에 가장 오래 붙잡히는 영수증 정리, 비용 분류, 장부 초안 작성, 홈택스 입력값 준비를 줄입니다.
