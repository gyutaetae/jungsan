# PRD: 정산

## Product

`정산`은 개인 소상공인/프리랜서를 위한 영수증/거래내역 정산 비서다. 영수증 사진과 거래내역 엑셀을 하나의 장부 데이터로 정리하고, 국세청 간편장부 엑셀과 필요경비 요약을 만든다.

## Goal

전국 아이디어톤 데모에서 "영수증과 거래내역이 장부가 된다"는 흐름을 짧고 선명하게 보여준다. 실제 서비스 완성도보다 데모 안정성, 이해 속도, 신뢰 가능한 자동화가 우선이다.

## Target Users

- 개인 소상공인
- 프리랜서
- 세무 신고를 직접 준비하지만 홈택스/장부 작업이 부담스러운 사용자

## Core Value

AI가 세무 신고를 대신하는 것이 아니라, 사용자가 검토할 수 있는 장부 초안을 빠르게 만든다. "모르는 값은 모른다고 표시"해 신뢰를 얻는다.

## MVP Scope

### In

- 영수증 촬영/이미지 업로드
- 여러 영수증 선택 후 1장씩 순차 분석
- 거래내역 엑셀/CSV 업로드
- 장부 검토 테이블에서 직접 수정
- 간편장부 엑셀 다운로드
- 필요경비 요약 화면 표시
- 샘플 데이터/샘플 추출 fallback
- Vercel 배포 가능한 Next.js 웹앱

### Out

- 홈택스 자동 로그인/자동 제출
- HWP 직접 생성
- 서버 DB
- 로그인/계정
- 계정과목 AI 추천
- 업로드 원본 파일 서버 저장
- 복잡한 세무 판단 자동화

## UX Principles

- 홈택스처럼 딱딱하지 않게 만든다.
- 단일 화면에서 끝낸다.
- 귀엽고 키치하되 세무 신뢰를 해치지 않는다.
- AI 결과는 항상 사람이 검토/수정한다.
- 데모 실패를 피하기 위해 fallback을 UI에 명시적으로 둔다.

## Success Criteria

- 영수증 이미지 또는 샘플 결과가 검토 테이블에 들어간다.
- 엑셀 업로드 행이 같은 검토 테이블에 합쳐진다.
- 사용자가 행을 수정할 수 있다.
- 확인 필요 항목이 있어도 경고 후 다운로드할 수 있다.
- `jeongsan-simple-ledger.xlsx`가 생성된다.
- 필요경비 요약이 화면에 표시된다.
