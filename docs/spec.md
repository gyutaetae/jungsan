# Spec Baseline

이 문서는 구현 phase가 빠르게 참조할 제품 기준 요약이다. 세부 요구사항은 [PRD](./prd.md), 사용자 흐름은 [flow](./flow.md), 데이터 규칙은 [data-schema](./data-schema.md)를 기준으로 한다.

## Service

- 서비스명: `정산`
- 한 줄 정의: 개인 소상공인/프리랜서가 영수증 이미지와 거래내역 파일을 검토 가능한 장부 초안으로 바꾸고, 간편장부 엑셀을 다운로드하는 웹 MVP.

## Target

- 개인 소상공인
- 프리랜서
- 세무 신고를 직접 준비하지만 홈택스/장부 작업이 부담스러운 사용자

## MVP Scope

포함 범위:

- 영수증 이미지 업로드 및 1장씩 순차 분석
- 거래내역 `.xlsx`/`.csv` 업로드
- 모든 입력을 `LedgerEntry[]` 검토 테이블로 병합
- 사용자의 장부 행 직접 수정
- 필요경비 요약 표시
- 간편장부 엑셀 다운로드
- 총수입금액 사용자 입력
- 홈택스 입력용 요약 표시
- 신고 준비 문서 검토용 xlsx 초안 다운로드
- 샘플 데이터와 샘플 추출 fallback

제외 범위:

- 홈택스 자동 로그인, 인증서 처리, 자동 제출
- 로그인, 계정, 서버 DB
- 업로드 원본 파일의 서버 저장
- HWP 직접 생성
- 최종 신고서류 생성
- 계정과목 AI 추천
- 복잡한 세무 판단 자동화

## Core Philosophy

- AI는 영수증에서 구조화 데이터를 추출하는 역할만 맡는다.
- 사람이 검토하고 수정하는 테이블이 제품의 신뢰 표면이다.
- 공급가액, 부가세 등 모르는 세금 값은 추정하지 않는다.
- 불명확한 값은 `vatStatus = "missing"` 및 확인 필요 상태로 남긴다.
- 데모 안정성을 위해 AI 또는 외부 자료 접근 실패 시에도 샘플 fallback으로 핵심 흐름을 보여준다.
- 신고 준비 문서는 검토용 초안으로만 제공하며, 홈택스 입력 전 사용자가 반드시 확인해야 한다.

## Success Criteria

- 영수증 이미지 또는 샘플 결과가 검토 테이블에 추가된다.
- 거래내역 파일의 행이 같은 검토 테이블에 병합된다.
- 사용자가 행을 수정할 수 있다.
- 확인 필요 항목이 있어도 경고 후 다운로드할 수 있다.
- `jeongsan-simple-ledger.xlsx`가 생성된다.
- 필요경비 요약이 화면에 표시된다.
- 사용자가 입력한 총수입금액과 장부 데이터를 바탕으로 홈택스 입력용 요약이 표시된다.
- `jeongsan-tax-prep-draft.xlsx` 검토용 초안이 생성된다.
